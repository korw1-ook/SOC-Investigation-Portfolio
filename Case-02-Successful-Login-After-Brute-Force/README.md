# SSH Authentication Investigation Case

This folder contains a complete investigation case related to SSH
authentication activity analyzed using Wazuh.

The documentation is split into multiple files for clarity.
Please read them in the order below.

---

## File Reading Order

1. **scenario.md**  
   Describes the incident scenario and objective.

2. **timeline.md**  
   Provides the chronological sequence of observed events.

3. **investigation.md**  
   Details the analysis of logs and findings.

4. **detection.md**  
   Explains how the activity was detected.

5. **iocs.md**  
   Lists indicators identified during the investigation.

6. **mitre.md**  
   Maps the activity to MITRE ATT&CK techniques.

7. **conclusion.md**  
   Summarizes the investigation outcome and lessons learned.

---

## Environment

- Kali Linux  
- Ubuntu Server  
- Wazuh SIEM  

---

## Notes

This case is created for learning and detection engineering practice.
